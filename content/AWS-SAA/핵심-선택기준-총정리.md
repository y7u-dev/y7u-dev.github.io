| 상황 | 솔루션 |
| --- | --- |
| 단일 EC2 전용 블록 스토리지 | EBS |

| 지속적 온프레미스 ↔︎ S3 연동 | S3 파일 게이트웨이 |

## 자주 나오는 함정 패턴

| 함정 | 올바른 판단 |
| --- | --- |
| 고속 인터넷 있는데 Snowball | 고속 인터넷 = Snowball 제외 |
| Parameter Store로 자동 회전 | 자동 회전은 Secrets Manager만 |
| CloudWatch SQL 쿼리 | CloudWatch는 전용 언어, S3 데이터 아님 |
| CloudTrail을 작업 대상으로 | CloudTrail은 API 로깅 전용 |
| Global Accelerator = CloudFront | GA는 캐싱 없음 |
| 단일 샤드로 초당 수만 건 처리 | 샤드 1개 = 초당 1,000건 한계 |
| CloudFront ACM 다른 리전 | CloudFront ACM은 반드시 us-east-1 |
| Lambda + RDS 연결 오류 | RDS Proxy 사용 |
| 예측 불가 트래픽에 예약 조정 | 예측 불가 → 동적 조정 |
